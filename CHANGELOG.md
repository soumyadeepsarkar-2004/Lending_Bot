# Changelog

All notable changes to the Lending_Bot project will be documented in this file.

## [0.1.0] - 2026-03-20

### Added
- Initial release of Lending_Bot contract
- Core smart contract implementation with deposit and withdrawal functionality
- Token integration for Stellar Asset Contracts (SAC)
- Persistent storage for user balances
- Admin initialization functionality
- Balance query functionality
- API documentation
- Development guide
- Deployment scripts for Stellar testnet

### Features
- Secure deposits and withdrawals
- Persistent accounting using Soroban storage
- Admin-controlled pool initialization
- Token transfer handling
- Balance tracking for lenders

### Documentation
- Comprehensive README with project description
- API documentation with all contract methods
- Development guide for setup and building
- Deployment instructions and scripts
- Contributing guidelines

### Technology
- Built with Rust and Soroban SDK v21.0.0
- Optimized for WASM compilation
- Compatible with Stellar Testnet (Protocol 22+)
- Secure authorization and authentication
